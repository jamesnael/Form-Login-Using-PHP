Tutorial Menggunakan Data PHP yang di clone
1.Pindahkan folder yang di clone ke folder XAMPP/htdocs
2.Setelah itu, buka PhpMyAdmin
3.Buat database baru dengan nama db_test
4.Import data db_test.sql yang terletak dalam folder ke database
5.Gunakan aplikasi dengan ketik di browser "localhost/(nama folder yang diclone tadi)"
6.Selesai, Aplikasi siap di gunakan
